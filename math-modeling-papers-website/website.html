COMAP Compass

Welcome to COMAP Compass! Here you can find previous Outstanding awarded papers from the annual COMAP math modeling competitions for undergraduate students.

Enter part or all of a keyword and a table will display with all of the papers that used that keyword below! If you'd rather see all papers at once, you can search without entering a keyword. There are also filters for the years and problem types to help narrow your search.

Note: This is not officially sanctioned by COMAP. This is a class project.
